# RenderFarm
Project to Cloud Computing and Virtualization course @ Instituto Superior Técnico

How run the project:
	1 	Run the script.sh in the root of project (Directory RenderFarm)
	1.1	Pass argument < 2 > to the script to compile, instrument and run the application with instrumented code
	1.2 Or pass argument < 3 > to the script to compile and run the application without instrumented code.

NOTE: script.sh has the same instructions in the header.
